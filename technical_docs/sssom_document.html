MappingSetDocument

class MappingSetDocument

A data class representing a single SSSOM document which is a holder for a MappingSet object plus a CURIE map.

mapping_set

The main part of the document: a set of mappings plus metadata. It is of type sssom_schema.MappingSet.

converter

An instance of curies.Converter, used to convert CURIEs in the mapping set.

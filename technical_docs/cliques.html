Utilities for identifying and working with cliques/SCCs in mappings graphs

This module contains several utilities for working with strongly connected components (SCCs) or cliques in mapping graphs. These utilities are used for identifying and manipulating SCCs in the context of data mappings.

Functions

to_digraph(msdf: MappingSetDataFrame) → 'networkx.DiGraph'

Convert a MappingSetDataFrame to a directed graph where the nodes are entities’ CURIEs and edges are their mappings.

Parameters:

msdf – A MappingSetDataFrame object.

Returns:

A networkx.DiGraph object.

split_into_cliques(msdf: MappingSetDataFrame) → List[MappingSetDocument]

Split a MappingSetDataFrame into documents corresponding to strongly connected components of the associated graph.

Parameters:

msdf – A MappingSetDataFrame object.

Returns:

List of MappingSetDocument objects representing strongly connected components.

Raises:

TypeError – If Mappings is not of type List.

group_values(d: Dict[str, str]) → Dict[str, List[str]]

Group all keys in the dictionary that share the same value.

Parameters:

d – A dictionary.

Returns:

A dictionary where values are grouped by keys.

get_src(src: str | None, curie: str) → str

Get the prefix of a subject/object in the MappingSetDataFrame.

Parameters:

  • src – Source.

  • curie – CURIE.

Returns:

Source prefix.

summarize_cliques(doc: MappingSetDataFrame) → pd.DataFrame

Summarize statistics on a clique document.

Parameters:

doc – A MappingSetDataFrame object.

Returns:

A DataFrame containing summary statistics for each clique.
